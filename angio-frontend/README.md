# angio-frontend

## Project setup
```bash
npm install
```

### Compiles and hot-reloads for development
```bash
npm run serve # run dev server on http://localhost:8081
```

### Compiles and minifies for production
```bash
npm run build
```

### Run your tests
```bash
npm run test
```

### Lints and fixes files
```bash
npm run lint
```

### Run your end-to-end tests
```bash
npm run test:e2e
```

### Run your unit tests
```bash
npm run test:unit
```
